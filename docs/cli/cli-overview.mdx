## Overview

The sunpeak CLI provides commands for managing MCP Apps, from project creation to local development and production builds.

## Commands

The CLI includes commands for project management and development:

| Command | Description |
|---------|-------------|
| `sunpeak new` | Create a new sunpeak project |
| `sunpeak dev` | Start the development server (includes inspector and MCP server) |
| `sunpeak inspect` | Inspect any MCP server in the inspector (standalone) |
| `sunpeak build` | Build resources and compile tools for production |
| `sunpeak start` | Start the production MCP server |
| `sunpeak upgrade` | Upgrade sunpeak to the latest version |

<Note>
The CLI is included when you install sunpeak. No separate installation required!
</Note>

## Development Workflow

The typical development workflow with sunpeak:

1. **Create** a new project with `sunpeak new`
2. **Develop** your resources using `sunpeak dev` for live preview in the inspector
3. **Test** your app with automated e2e tests using the inspector (`pnpm test:e2e`)
4. **Visual test** for regressions with `sunpeak test --visual` (see [Testing Guide](/mcp-apps-testing#visual-regression-testing))
5. **Live test** against real ChatGPT with `pnpm test:live` (see [Testing Guide](/mcp-apps-testing#live-testing))
6. **Build** for production with `sunpeak build`
7. **Start** the production server with `sunpeak start`
